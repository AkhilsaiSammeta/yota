/*
 * ArrayList<T>
 * - Flexible array that grows automatically
 * - Access: O(1) - instant by index
 * - Insert: O(N) worst case - but append is O(1)
 * - Used for: attributes, instances, distances
 * 
 * HashMap<K, V>
 * - Stores key-value pairs
 * - Lookup: O(1) average - very fast
 * - Used for: frequency counts, voting
 * 
 * Arrays (int[], double[], Object[])
 * - Fixed size, simple
 * - Access: O(1)
 * - Used for: distances, sorting
 */

// ====================================
// REFERENCE: ALGORITHMS USED
// ====================================

/*
 * SORTING: Bubble Sort
 * - Simple, easy to understand
 * - Time: O(N^2) but N is small (K neighbors)
 * - Code: Simple nested loops
 * 
 * DISTANCE: Euclidean Distance
 * - Standard in ML
 * - Formula: sqrt(sum((a[i] - b[i])^2))
 * - Easy to understand
 * 
 * CLASSIFICATION: K-Nearest Neighbors
 * - Simple algorithm for beginners
 * - Lazy learning (no training needed)
 * - Vote based on K closest neighbors
 */
